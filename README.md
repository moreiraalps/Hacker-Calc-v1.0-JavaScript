Uma calculadora web com estética **Cyberpunk/Hacker**, focada em alta performance e design de terminal retrô. Desenvolvida com JavaScript puro, explorando manipulação de DOM e CSS Grid avançado.

> "I'm in." — Acesso concedido ao sistema de cálculos de alta precisão.

---

## 🖥️ Preview

A interface utiliza uma paleta de cores `#00ff41` sobre preto absoluto, simulando monitores CRT antigos com efeitos de *glow* (brilho) e fontes monoespaçadas.

## 🛠️ Tecnologias de Invasão

* **HTML5 Semântico:** Estrutura robusta e limpa.
* **CSS3 (Neon FX):** Uso de `box-shadow` e `text-shadow` para o efeito Matrix.
* **Vanilla JavaScript:** Lógica de cálculo customizada com tratamento de dízimas periódicas.
* **Google Fonts:** `Roboto Mono` para a estética de terminal.

## 🚀 Funcionalidades Nucleares

- [x] **Grid 5-Columns:** Layout fiel às calculadoras de mesa profissionais.
- [x] **Overflow Protection:** Os números nunca quebram o visor (limite de 10 dígitos).
- [x] **Smart Rounding:** Resultados complexos são arredondados automaticamente.
- [x] **Terminal Header:** Simulação de prompt de comando `root@moreiraalps:~#`.

## 📦 Como Clonar e Executar

1.  Abra o seu terminal.
2.  Clone este repositório:
    ```bash
    git clone [https://github.com/moreiraalps/hacker-calc-js.git](https://github.com/moreiraalps/hacker-calc-js.git)
    ```
3.  Entre na pasta:
    ```bash
    cd hacker-calc-js
    ```
4.  Abra o `index.html` no seu navegador.
